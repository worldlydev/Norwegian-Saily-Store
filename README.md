# Cooking Development Installation  

- Install cocoaPods  
- (Optional) Comment this line -> # source 'https://cdn.cocoapods.org/'  
- pod install  
- Build it and run  

---> Daemon may not run on macOS or and x86 devices, and required to comment some lines in build scripts.
